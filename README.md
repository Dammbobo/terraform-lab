# Terraform Lab

This repository demonstrates basic Infrastructure as Code (IaC) concepts
using Terraform.

## Why this project
Terraform allows infrastructure to be defined, versioned, and deployed
consistently. This lab shows how Terraform manages resources declaratively.

## What this project does
- Configures a Terraform provider
- Creates a managed local resource
- Uses variables and outputs

## Tech Stack
- Terraform
- Infrastructure as Code (IaC)

## How to run locally

```bash
terraform init
terraform plan
terraform apply
